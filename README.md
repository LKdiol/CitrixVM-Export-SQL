# CitrixVM-Export-SQL
기존 카탈로그 시스템에서 VM정보를 추출하는 쿼리

#사용방법
- 1. VMExport.sql,VMExport2.sql 선택하여 SQL Server Management Studio에서 쿼리를 붙여넣은 후 결과 값을 모두선택(Ctrl + A) 후 다른이름으로 결과 저장 하여 csv형식으로 저장
- 2. 생성된 CSV파일을 DDC서버로 옮긴 후 해당 csv파일 불러오기 진행
- 3. 정상적으로 불러온 것을 확인
